# 🎬 Modelado de Base de Datos para un Sistema de Cine

Proyecto académico enfocado en el análisis y diseño de una base de datos para un sistema de gestión de cine, aplicando el proceso de normalización y el modelado mediante diagramas Entidad-Relación (E-R).

## 📖 Descripción

El objetivo de este proyecto fue diseñar una base de datos organizada y libre de redundancias a partir de un caso de estudio de un cine.

Durante el desarrollo se aplicaron los principios de normalización hasta la **Tercera Forma Normal (3FN)** y posteriormente se construyeron los diagramas necesarios para representar la estructura de la base de datos.

## 📄 Desarrollo del proyecto

### 1️⃣ Normalización

Se realizó el proceso de normalización de una base de datos siguiendo las tres primeras formas normales:

* **1FN:** separación de atributos no atómicos y organización de la información.
* **2FN:** eliminación de dependencias parciales.
* **3FN:** eliminación de dependencias transitivas.

---

### 2️⃣ Modelo Entidad–Relación

Con las entidades obtenidas durante la normalización se diseñó un diagrama conceptual definiendo las relaciones y sus cardinalidades.

---

### 3️⃣ Modelo UML

Finalmente se representó el modelo de datos mediante un diagrama UML que refleja las entidades, atributos y relaciones obtenidas durante el análisis.
